# HermesX

HermesX 是你的本地 AI 工作中枢。它不止回答问题，而是理解目标、接住上下文、拆解任务、处理文件，并把每一步进展清楚地呈现在你眼前，让灵感、资料与成果在同一个空间自然流动。

你只需要说出想要抵达的结果，HermesX 会在对话、文件和任务之间建立秩序，把复杂工作收束为可追踪、可掌控、可交付的过程。你负责判断方向，它负责推进过程；最终留下的不只是答案，而是一份可打开、可定位、可复盘的完成品。
