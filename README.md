# HermesX

本地智能工作台 — 把 AI Agent 装进你的电脑。支持 Windows 和 macOS（Apple Silicon）。

> 下载地址：[hermesx.jackcloud.online](https://hermesx.jackcloud.online)

---

## 更新日志

### v0.3.1（2026-06-19）

Agent 头像系统重新设计，工具执行器加固，宣传页迁移到管理后台仓库统一管理。

### v0.3.0（2026-06-19）

对话中生成的文件（网页、文档等）会自动识别为「产物」，集中展示在工作区侧边栏。Agent 启动时会真正读取项目里的约定文件（如 `AGENTS.md`），而不是只在系统提示里说说。新增文件追加写入能力，构建大文件更安全。Token 用量面板和跨会话学习也有改进。

### v0.2.9（2026-06-19）

加了多层安全防护：工具调用的写入闸门、产物输出校验、权限选择器。Agent 执行前会检查文件操作的安全性，避免误删或写到不该写的地方。新增自动恢复和项目续做能力，任务中断后能接着干。

### v0.2.8（2026-06-18）

工具执行器加固，Local Agent 主循环优化，增量迭代预算控制。跨会话经验学习功能上线 — Agent 能记住之前踩过的坑。

### v0.2.6（2026-06-18）

新增敏感信息自动脱敏，输入框里的密码、密钥不会原样发给模型。指令面板大幅简化，任务规划逻辑更新。

### v0.2.5 及更早

桌面宠物系统、多模型切换、本地文件读写、终端命令执行、MCP 工具生态接入、备份恢复等基础能力陆续上线。
