# HermesX

**本地优先的 AI 工作中枢，让模型不止于回答，而是真正把事情做完。**

HermesX 理解你的目标，接住散落的上下文，自主拆解任务、调度工具、读写文件，并把每一步推进透明地呈现在你眼前，让灵感、资料与成果在同一个空间里自然流动。

你只需说出想要抵达的结果，它便在对话、文件与任务之间建立秩序，把复杂工作收敛为可追踪、可掌控、可交付的过程。你负责判断方向，它负责推进落地；最终交付的不只是一段答案，而是一份可打开、可定位、可复盘的成品。数据与执行皆在本地完成，安全与掌控，始终在你这一侧。

## 版本日志

### v1.0.4（2026-06-24）

- 多模型文生图：DashScope 千问图像生成 + 多适配器扩展，Agent 可调用不同模型出图
- 生成图画廊重构：单图完整展示 + 多图方格画廊，视觉体验大提升
- 专利交底书起草技能：从草图到专利交底书，Agent 一键生成
- 跨OS命令环境感知：Agent 自动识别系统差异，命令适配更精准
- 跨会话学习纠偏：文生图选错模型不会再被学成「忌讳」
- 引擎面板修复：新任务不再残留旧步骤，状态管理更干净

### v1.0.3（2026-06-23）

- 图片生成：Agent 可直接生成图片并嵌入对话，支持多模型适配器
- Markdown 渲染增强：图片内联展示、格式更丰富
- CommandBar 交互优化
- HTTP 代理与 Provider 增强

### v1.0.2（2026-06-23）

- PPT 生成：Agent 可直接产出演示文稿（HTML Deck + PPTX），从需求到成品一气呵成
- 计划提案：复杂任务先出方案再执行，每一步有据可查
- Provider 流控：Rate Limit Gate 防止 API 调用过载
- RAG 引擎增强：关键词检索更精准
- Tool Executor 重构：工具注册中心 + 安全守卫，执行更稳健

### v1.0.1（2026-06-22）

从「只读分析」到「亲自动手」——这一版让 Agent 获得了看屏幕和操作桌面的能力：

- 看屏幕 · 操作桌面：截屏感知画面，用鼠标键盘操控任意桌面应用与浏览器——打开网页、点击按钮、填写表单、启动软件，像人一样在现场动手
- 子任务并行协作：多个子 Agent 围绕同一现场并行推进，过程实时回流
- Windows 无障碍增强：UIA 元素定位与窗口遍历更稳定
- 浏览器捕获深化：Console / Network 事件常驻监控，网页交互更精准
- 权限面板重构：操作权限更清晰可控

让 Agent 不再只是"说"，而是真正"做"。

从 0.x 一路打磨到 1.0.0，HermesX 正式开启公测。这一版我们把重心放在「让 Agent 更自主、更安全」上：

- 子 Agent 调度：自动拆解复杂任务，派生子 Agent 并行推进

- 灾难防护：关键删除操作可撤销，误删有后悔药
- 安装包完整性校验 + 联网安全防护
- MCP 生态接入：GitHub / PostgreSQL 等外部工具即插即用
